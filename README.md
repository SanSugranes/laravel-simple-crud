# laravel-simple-crud
Simple CRUD app demo using Laravel



### Working env, stack and techs
- Working on Win 11 VM
- XAMPP
  - Apache
  - MySQL
  - PHP
- Node
- Composer
- Laravel
- [Tailwind]

### Features
- (CRUD) Students
  - Create
  - Read
  - Update
  - Delete
- [Login]
  - Admin
    - CRUD all
  - Student
    - Read only
